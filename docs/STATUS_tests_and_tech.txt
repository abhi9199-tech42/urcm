API Limits
- LLMBridge: circuit breaker with env-tunable thresholds LLM_BREAKER_THRESHOLD, LLM_BREAKER_COOLDOWN.
- Model integrity: set LLM_MODEL_SHA256 to enforce GGUF checksum before loading.
- Executive explain: bounded chain depth (≤6) to prevent long traversals.
- Ingestion: STOP_WORDS filter prevents attractor collapse; relations parsed for basic forms only.

Failure Modes
- Missing local LLM: bridge enters MOCK mode; responses are tagged with [MOCK].
- Integrity mismatch: bridge refuses model load; system continues without LLM.
- Dimension mismatch: brain reload resets state to neutral; re-ingest to rebuild.
- Convergence stall: metacognitive noise injection; check μ diagnostics.

Incident Response Playbooks
- Unsafe prompt handling: verify refusal in demo, review logs/unsafe_prompts.log, adjust values if required.
- Model asset issues: recompute SHA256, update LLM_MODEL_SHA256, restart app.
- Performance regressions: run performance tests, inspect recent changes to smoothing and gating.
- Logic inconsistencies: use Executive explain to surface chains and contradictions; re-ingest corrected knowledge.
